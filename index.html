Quick Guide

1. Upload count matrix → pick gene ID column → select samples.


2. Pick species → upload codon CSV → confirm its gene column.


3. Run — usage / proportion / RSCU bias + PCA. Optionally normalize to FC vs reference.


4. Download CSVs or ZIP.

FC Normalization

In the Results tabs you can pick a reference: a single sample or the mean of multiple samples.


Display values become:

Usage / Proportion:
log₂(sample / ref)
Bias (RSCU−1):
sample − ref

Raw data is always preserved for download.
